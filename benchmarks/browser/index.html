Methodology: Renders N rects on a 1280×720 WebGL2 canvas (Skia via CanvasKit). First 60 frames discarded (warm-up). Then 5 s of frames recorded. Static — objects don't move; raw GPU pixel throughput. Animated — every object moves each frame; mutation + re-render throughput. p5 FPS — 5th-percentile frame rate (the worst 5% of frames). Higher is always better.
